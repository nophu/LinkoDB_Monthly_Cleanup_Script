How this script will work: 
What is needed for the script to work? **Please have the rubric file (ideally an Excel File for what needs to be checked), the Excel sheets from the Linko DB that needs to be cleaned up & the email that the generated summary report needs to be sent to.**

1) There will be a page that is open for the script, it will prompt you to choose two file procedures and one procedure asking for the email:
                       * One slot for the rubric file
                       * One slot for ALL data sheets that need to be checked
                       * One textbox for your email, preferrably please use the email given to you by the City of Utilities.
   
2) After inputting ALL files, click generate button and a summary report of what changes needed to be made will sent out in an email to you, the user.

3) Please look over the Summary report and refer to the data sheets, as it will tell you where each change needs to be made and if the changes can be made. IF needed, there will also be a modified report of the original data sheets to ensure proper process.

4) Lastly, please leave any feedback needed if the process was too complicated and you are in need of assistance. Please and thank you!
